Chapter 5 MS2-Annotation

This chapter consists of: 1. Submit job of CMN and jobs based on CMN on GNPS (involved uploading files to the server of GNPS) 2. Submit job of FBMN and jobs based on FBMN on GNPS (involved uploading files to the server of GNPS) 3. Compare MS2-experimental data with our in-house library 4. Download all results from all submitted jobs 5. Merge the results from each job
